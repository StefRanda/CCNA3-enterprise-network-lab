# CCNA 3 - Enterprise Networking, Security and Automation (ENSA)

## 📌 Project Overview

This repository contains my **CCNA 3 Integrative Lab**, developed in Cisco Packet Tracer.

The objective of this project is to simulate an enterprise network applying the technologies learned throughout the CCNA Enterprise Networking, Security and Automation course.

⚠️ **Current status:** Approximately **90% completed**. This is an intermediate version of the final topology and will continue receiving updates.

---

## Technologies Implemented

- VLANs
- Inter-VLAN Routing
- DHCP
- OSPF
- BGP
- WAN Links
- Internet connectivity simulation
- Wireless LAN Controller (WLC)
- Access Point integration
- Layer 2 & Layer 3 Switching
- Static and Dynamic Routing
- Basic Network Security (in progress)
- Troubleshooting and verification commands

---

## Objectives

- Design a scalable enterprise network.
- Implement routing using OSPF and BGP.
- Simulate Internet connectivity.
- Provide IP addressing using DHCP.
- Deploy wireless infrastructure through a Cisco WLC.
- Practice troubleshooting techniques similar to real production environments.
